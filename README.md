# bedrock-unicode-characters
**Minecraft: Bedrock Edition** символы Unicode

## Как использовать
Скопируйте символ Unicode и вставьте его в чат Minecraft: Bedrock Edition, на таблички, в заголовки и т.д.

## Как это работает?
Minecraft использует пакеты ресурсов для назначения символов (глифов) различным значениям Unicode, что позволяет игре отображать текст в чате и на табличках. Поскольку шрифты в пакетах ресурсов представляют собой просто текстуры, игра также может добавлять "эмодзи", которые имеют свои собственные значения Unicode, назначенные им так же, как и буквам.
